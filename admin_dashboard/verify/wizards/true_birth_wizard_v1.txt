# AI 自由提问区
当你（Child AI）发现某一宫位对应反应强烈、或人生事件时间点可判别「出生时辰」时，你可以提出额外 1-3 个针对性追问，用于锁盘。

追问原则：
- 只问可以“验证时辰”的问题
- 不问性格、不问抽象哲学
- 追问必须可以用日期 / 事件量化
